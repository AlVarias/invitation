# invitation
Allen & Marry's Wedding Invitation

An interactive, animated, and mobile-responsive React web application created for our wedding on September 8, 2026. Features a 3D CSS folded envelope animation, a live countdown, embedded Google Maps navigation, and a fully functional RSVP system.

🌟 Features

Dramatic Entrance: Custom 3D envelope and folded card reveal animation using Tailwind CSS and React state.

Live Countdown: Real-time countdown to the wedding date.

Interactive RSVP: Conditional logic form that submits directly to email via FormSubmit.

Map Integration: Embedded maps with quick links to Google Maps and Waze.

Spring Garden Theme: Custom color palette featuring Blush, Sage, Dusty Rose, Mist, and Soft Gold.

🛠️ Tech Stack

React (Vite or Create React App)

Tailwind CSS (for styling and animations)

Lucide React (for minimalist SVG icons)

FormSubmit (for backend-less RSVP email forwarding)

🚀 Getting Started

Prerequisites

Make sure you have Node.js installed on your computer.

Installation

Clone the repository

git clone [https://github.com/yourusername/wedding-invitation.git](https://github.com/yourusername/wedding-invitation.git)
cd wedding-invitation


Install dependencies

npm install
npm install lucide-react


Install Tailwind CSS (if not already configured)
Follow the Tailwind CSS Vite guide.

Run the development server

npm run dev
